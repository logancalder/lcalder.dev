  • // Description
  • Over the summer of 2023, I was fortunate enough to intern at Hewlett Packard Enterprise, working as a software engineer in their Aruba Networks divison.
  • My work mainly consisted of memory and runtime bug fixes, as well as developing an interactable backend platform used by developers, sales engineers, customer service, and quality assurance workers.
  • Sadly, I cannot showcase my work as it was protected by HPE and confidential to the public. I will, however, go into detail of my internship experience, and what I learned along the way.


  • // My Projects
  • My first project was to resolve unique device IDs and other data not properly being inserted into our Redis database. This had been leading to corrupted data, as well as excessive memory usage.
  • To fix this, I learned Golang---a language I had prior no knowledge of working with. I also picked up Docker and Kubernetes to manage image deployment, as well as Git through Jira's Bitbucket. I am pleased to say that I was able to resolve the issue, which in some instances conserved up to 80% more memory.
  • My second project was to work on a fullstack application that would permit HPE employees to lookup & manage device data, such as IDs, paired devices, and routing information. Prior, the website that they had been using was dated, and extremely not user friendly. Searching for specific data had to be manually entered, or typed in with no aid from the website.
  • Along with another intern, we managed to implement a user-friendly UI that automatically filled out redundant information, and would display device IDs in a more readable format.


    • // Experience
    • My internship was not only extremely enjoyable and fun, but also highly benefitial for my programming knowledge & professional workplace experience. Some key takeaways that I got from my time at HPE were learning collaborative development and deployment, image scaling using Docker and Kubernetes, and fullstack developing by learning Golang and Javascript, as well as suplementing my pre-existing knowledge of Python, HTML, & CSS.
    • I greatly enjoyed working alongside experienced developers, and learning from their advice. I would now fully consider myself to be a much better programmer than I was before, coding more efficiently and professionally than I had prior. Since my time at HPE, I have been able to take my new skills back to personal projects and academic ones, leading groups in developing projects, as well as managing deployments to live products.
