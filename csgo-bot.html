  • String lang = "py";
  • // Description
  • CSGO Bot is an interfaceable bot useable in Discord that mimics the in-game market of Counter Strike: Global Offense.
  • The bot allows for users to buy and open crates, sell, display, or showcase items, and view other users profiles.
  • This bot is not available for download, but is accessible on the CSGO Bot Discord server


  • // Features
  • This bot is home to many features, with the core of them being focused around replicating the in-game skin trading market in CS:GO.

    • Case purchasing
    • Bulk Case Opening
    • Bulk Skin Selling
    • Per-Skin IDs
    • ID Statistic Analysis
    • Inventory Mangement
    • Money Balance
    • Showcase #1 Skin
    • Per-User Profiles
    • Coin Flip Gambling


  • // Development
  • This project was my first major discord.py application, which took around 30 hours and 2000 lines of code to develop. Having to self-learn the discord.py library, I found this project as not only a fun project, but a challenge as to how I could push myself to better my Python knowledge and skills.
  • I found it fun learning a new library, as well as a bunch of new Python knowledge. Reading up on how different methods worked on Reddit, watching YouTube videos, and getting advice from fellow developers on Discord were the main resources I ended up using.
  • The main methodology behind this program was nested lists, which allowed me to store a vast multitude of data in groups. /* I did not use dictionaries as I found them less efficient for the usage I needed */ I also learned to read and write data to files in order to contain per-user data.
  • I also needed to find a way to host my bot. While cloud services such as AWS are an option, they cost money and I wasn't too appealed at the idea. To provide myself a solution to this issue, and to prevent the need to purchase more cloud hosting services in the future, I purchased a Raspberry Pi and learned to host applications off my own server.
  • The addition of the Raspberry Pi caused me to learn some more technical aspects of computing that I was previously unaware of, such as interacting with the Command Prompt to push and pull programs from GitHub, and to download new releases of coding languages and libraries.


  • // Usage
  • This bot utilizes the prefix !cs {message}.
  • To learn more about the individual commands that this bot offers, run the command !cs help.
